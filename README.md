# Install

* Copy zarch.zip to Desktop .
* Open The Terminal .
* Type the following commands :

    - [X] cd Desktop
    - [X] unzip sarch.zip
    - [X] cd zarch
    - [X] chmod +777 sarch.sh
    - [X] ./sarch.sh  or  bash sarch.sh
